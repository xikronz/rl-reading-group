# Awesome-rl-reading-group

Welcome to CUAI's RL/interactive learning reading group website + materials :)

📄 **Want us to read a specific paper?** [Open an issue](https://github.com/xikronz/awesome-rl-reading-group/issues/new) with your suggestion, we're always taking paper requests!


# Syllabus 

The schedule for the summer is still being finalized but broadly speaking we want to cover the following:
1. online learning algorithms and games
2. sequential decision making and MDPs
3. policy gradients 
4. RL for LLMs
5. RL for agents 
6. RL for consistency models

and make time for our amazing line up of guest speakers from Cornell, MIT, Stanford as well as industry: Mercor, Citadel Securities and More!
